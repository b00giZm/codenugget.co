# codenugget.co

The code which powers the upcoming version of [codenugget.co](https://codenugget.co).

## Getting Started

### Requirements

* `node` v12.16 or higher
* `npm` v6.14 or higher

### Development Server

```
npm install
npm run dev
```

Will start a development server at [http://localhost:3000](http://localhost:3000).
